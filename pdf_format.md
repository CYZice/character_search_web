# PDF 文件格式描述

## 整体布局

本 PDF 为汉字检索文档，每页采用**双栏（两列）布局**。

```
┌─────────────────────────────────────┐
│           第 N 页                    │
├───────────────┬─────────────────────┤
│               │                     │
│   左列        │        右列         │
│  (Column 1)   │     (Column 2)      │
│               │                     │
└───────────────┴─────────────────────┘
```

## 单列内部结构

每一列的构成如下：

```
┌─────────────────┐
│    【字】       │  ← 标题字（被【】框住）
│                 │
│   ┌───┐        │
│   │图1│        │  ← 图片竖排排列
│   └───┘        │
│   ┌───┐        │
│   │图2│        │
│   └───┘        │
│   ┌───┐        │
│   │图3│        │
│   └───┘        │
│    ...         │
│                 │
└─────────────────┘
```

### 结构说明

1. **标题字**：位于列顶部，用【】框住，表示当前列所有图片对应的汉字
2. **图片区**：标题字下方，连续竖排排列的图片
   - 排列方向：从上到下
   - 所有图片都属于该标题字

## 阅读顺序

PDF 的阅读顺序按以下规则：

```
1. 左列：从上到下读完
       ↓
2. 右列：从上到下接着读
       ↓
3. 下一页：重复 1-2
```

### 具体流程

```
┌─────────────────┬─────────────────┐
│  【字A】        │  【字B】        │
│   图1           │   图1           │
│   图2           │   图2           │
│   图3           │   图3           │
│   ...           │   ...           │
│   图N           │                 │ ← 左列读完
│                 │                 │
├─────────────────┼─────────────────┤
│  【字C】        │  【字D】        │ ← 换到右列
│   图1           │   图1           │
│   ...           │   ...           │
└─────────────────┴─────────────────┘
```

## 页面流转示意

假设有 4 个字（甲、乙、丙、丁），每列最多容纳 N 个图片：

| 页码 | 左列 | 右列 |
|------|------|------|
| 第 1 页 | 【甲】图 1~N | 【乙】图 1~M |
| 第 2 页 | 【丙】图 1~K | 【丁】图 1~L |

---

## 关键特征总结

- **每页列数**：2 列
- **每列结构**：【标题字】+ 连续图片
- **图片排列**：竖排（从上到下）
- **页面顺序**：左列优先，右列次之
- **字-图关系**：一个【标题字】对应多张图片
